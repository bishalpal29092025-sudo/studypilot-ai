# 🚀 StudyPilot AI

> The AI that learns how **YOU** study.

StudyPilot AI is a production-grade AI-powered personalized study coach that dynamically adapts learning plans, quizzes, revisions, and study strategies based on user performance.

---

## ✨ Features

- 🎯 Personalized onboarding
- 🧠 AI roadmap generation
- 💬 AI study companion
- 📝 Adaptive quizzes
- 📈 Progress dashboard
- 🔥 Study streak tracking
- 📚 Weak topic detection
- 🔁 Spaced repetition
- 📄 PDF roadmap export
- 🌙 Dark mode
- 📱 Fully responsive UI

---
